# Beleg

**Definition:** Ein digitalisiertes Buchungsdokument (Rechnung, Quittung, Kassenbon, Bewirtungsbeleg), das ueber Threema oder PDF-Import ins System gelangt.

**Beispiel:** Mandant fotografiert eine Restaurantrechnung und sendet sie per Threema. Das System extrahiert per OCR Betrag, Datum, Lieferant und schlaegt eine SKR04-Kontierung vor.

**Gegenbeispiel / Abgrenzung:** Ein Angebot oder Kostenvoranschlag ist kein Beleg — es fehlt der Zahlungsnachweis. Interne Notizen sind ebenfalls keine Belege.

**Regulatorik:** GoBD (§146 AO, §257 HGB) — Belege muessen unveraenderbar, vollstaendig und zeitnah erfasst werden. Aufbewahrungsfrist: 10 Jahre.
